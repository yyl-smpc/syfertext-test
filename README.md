## 安装syferttext
---
### 1.1激活pysyft环境
```
conda activate pysyft
```

### 1.2克隆syfertext
```
git clone git@github.com:OpenMined/SyferText.git
```
### 1.3cd到syfertext项目目录下

### 1.4切换分支并安装

```
git checkout update_additive_sharing
python setup.py install
```

## 启动测试代码
---

```
python syfertext-test.py
```
